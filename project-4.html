Project Overview

RevShop is now refactored into a robust microservices architecture to enhance scalability, fault tolerance, and high availability. The project utilizes Spring Cloud Gateway for routing, Eureka for service discovery, Resilience4j for circuit breaking, and Spring Cloud LoadBalancer for distributing traffic across multiple instances. The microservices architecture consists of independent services handling core functionalities such as products, orders, and payment processing. The deployment is done on AWS using Docker and Kubernetes, and monitoring is powered by Prometheus. Jenkins ensures smooth CI/CD pipelines for automation.

The goal is to provide a highly scalable and fault-tolerant e-commerce solution that can handle growing traffic and ensure seamless operation even in case of failures.
